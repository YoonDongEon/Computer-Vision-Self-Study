# 자동 문서 스캐너

## 프로그램 개요(필요성)

이 프로그램은 사진으로 촬영한 문서, 보고서, 책 페이지, 영수증처럼 사각형 형태의 문서를 자동으로 검출하고, 정면에서 스캔한 것처럼 보정해주는 컴퓨터 비전 기반 GUI 프로그램입니다. 이미지 파일뿐 아니라 PDF 문서의 각 페이지도 불러와 같은 방식으로 보정 결과를 확인할 수 있습니다.

실제 문서를 스마트폰이나 카메라로 촬영하면 기울어짐, 원근 왜곡, 배경 포함, 조명 불균형 때문에 바로 사용하기 어려운 경우가 많습니다. 특히 문서 정리, PDF 변환, OCR 전처리, 과제 자료 정돈 같은 작업에서는 문서 외곽을 정확히 찾아 반듯하게 펴는 과정이 중요합니다. 이 프로그램은 문서 영역을 자동으로 찾아 투시 변환을 적용하고, 필요하면 꼭짓점을 직접 수정할 수 있게 하여 자동화와 수동 보정의 장점을 함께 사용할 수 있도록 구성했습니다.

## 프로그램 기능 및 사용법 설명

### 1. 주요 기능

- 문서 자동 검출
  - 이미지에서 가장 유력한 문서 외곽선을 찾아 4개의 꼭짓점을 자동으로 설정합니다.
- PDF 페이지 불러오기
  - PDF 문서를 열고 현재 페이지를 이미지처럼 렌더링하여 자동 검출과 보정 작업을 진행할 수 있습니다.
- PDF 페이지 이동
  - 여러 페이지로 구성된 PDF에서 이전 페이지, 다음 페이지 버튼으로 페이지를 바꿔가며 스캔할 수 있습니다.
- 꼭짓점 드래그 보정
  - 자동 검출 결과가 완벽하지 않을 경우, 원본 이미지 위의 점을 직접 드래그해 문서 경계를 수정할 수 있습니다.
- 투시 변환 기반 스캔
  - 선택된 4개 꼭짓점을 기준으로 문서를 정면 시점 이미지로 보정합니다.
- 출력 모드 선택
  - 컬러, 그레이스케일, 흑백 모드로 결과를 확인할 수 있습니다.
- 문서 후처리
  - 밝기, 대비, 선명도, 흑백 임계값을 조정해 가독성을 높일 수 있습니다.
- 결과 저장
  - 최종 스캔 결과를 `PNG`, `JPG`, `BMP`, `PDF` 형식으로 저장할 수 있습니다.

### 2. 실행 방법

이 프로그램은 Python 기반 GUI 프로그램이며, 실행 전 필요한 라이브러리가 설치되어 있어야 합니다.

#### 실행에 필요한 라이브러리

- `opencv-python`
- `numpy`
- `Pillow`
- `PyMuPDF`

#### 실행 명령

터미널에서 아래와 같이 실행합니다.

```bash
cd Computer-Vision-Self-Study/1_Automatic-Document-Scanner
python DocumentScanner.py
```

필요 라이브러리가 없다면 아래와 같이 설치할 수 있습니다.

```bash
pip install -r requirements.txt
```

### 3. 프로그램 사용 순서

#### 1) 이미지 열기

- 상단의 `이미지/PDF 열기` 버튼을 눌러 문서가 포함된 이미지 또는 PDF 파일을 선택합니다.
- 이미지를 불러오면 프로그램이 자동으로 한 번 문서 검출을 시도합니다.
- PDF를 불러오면 첫 페이지가 열리며, 페이지 버튼으로 다른 페이지로 이동할 수 있습니다.

#### 2) 문서 자동 검출 확인

- `문서 자동 검출` 버튼을 누르면 문서 외곽 4점이 자동으로 선택됩니다.
- 선택된 영역은 원본 이미지 위에 오버레이 형태로 표시됩니다.
- PDF 페이지도 동일한 방식으로 자동 검출됩니다.

#### 3) 꼭짓점 수동 보정

- 자동 검출된 결과가 부정확하면, 원본 이미지 위의 점을 마우스로 드래그해 위치를 조정합니다.
- 자동 검출이 실패한 경우에는 `점 초기화` 후 이미지 위를 직접 클릭해 4개의 꼭짓점을 수동으로 지정할 수도 있습니다.
- 점 순서는 좌상단, 우상단, 우하단, 좌하단 기준으로 정렬됩니다.

#### 4) 스캔 실행

- `스캔 실행` 버튼을 누르면 선택된 문서 영역이 정면 이미지로 변환됩니다.
- 변환 결과는 오른쪽 미리보기 캔버스에 표시됩니다.

#### 5) 후처리 조정

- 우측 설정 패널에서 출력 모드와 보정값을 조절할 수 있습니다.
- `컬러` 모드
  - 원본 색상을 유지한 문서 스캔 결과를 확인할 수 있습니다.
- `그레이스케일` 모드
  - 색상을 제거하고 명암 중심으로 문서를 정리할 수 있습니다.
- `흑백` 모드
  - 흑백 임계값을 기준으로 명확한 문서 형태를 만들 수 있어 OCR 전처리에 유리합니다.
- 밝기, 대비, 선명도 슬라이더를 조정하면 결과 미리보기에 반영됩니다.
- `후처리 적용` 버튼으로 현재 설정을 다시 적용할 수 있으며, `설정 초기화` 버튼으로 기본값으로 되돌릴 수 있습니다.

#### 6) 결과 저장

- `결과 저장` 버튼을 누르면 현재 미리보기 상태의 스캔 결과를 파일로 저장할 수 있습니다.
- 입력 파일이 이미지이든 PDF이든 관계없이 결과는 `PNG`, `JPG`, `BMP`, `PDF` 중 원하는 형식으로 저장할 수 있습니다.
- PDF로 저장하면 현재 스캔 결과가 1페이지짜리 PDF 문서로 저장됩니다.

### 4. 사용 시 참고 사항

- 문서 외곽과 배경의 대비가 뚜렷할수록 자동 검출 정확도가 높아집니다.
- 문서가 심하게 가려져 있거나 배경과 섞여 있는 경우에는 자동 검출이 실패할 수 있습니다.
- 자동 검출 실패 시 드래그 보정 또는 수동 4점 지정으로 계속 작업할 수 있습니다.
- 흑백 모드는 문서 텍스트를 선명하게 만들 수 있지만, 사진이나 컬러 정보가 중요한 문서에는 적합하지 않을 수 있습니다.
- PDF는 페이지를 이미지로 변환한 뒤 처리하므로, 매우 큰 문서나 페이지 수가 많은 파일에서는 로딩 시간이 길어질 수 있습니다.
