# 투시변환 이미지 처리기

## 프로그램 개요

이 프로그램은 사진 속 문서, 안내판, 모니터 화면, 표지판처럼 비스듬하게 촬영된 사각형 영역을 정면에서 본 것처럼 보정하기 위한 투시변환(Perspective Transform) GUI 프로그램입니다.

일반적인 사진에서는 촬영 각도 때문에 원근 왜곡이 발생하여, 실제로는 직사각형인 대상이 사다리꼴처럼 보이는 경우가 많습니다. 이런 왜곡은 문서 스캔, 자료 정리, 화면 캡처 보정, 이미지 전처리 작업에서 불편을 만들 수 있습니다. 이 프로그램은 사용자가 직접 4개의 꼭짓점을 지정해 원하는 영역만 정사각형 또는 직사각형 형태로 펼칠 수 있도록 도와주며, 변환 후 밝기, 대비, 선명도까지 한 번에 조정할 수 있도록 구성되어 있습니다.

## 프로그램 기능 및 사용법 설명

### 1. 주요 기능

- 이미지 파일 불러오기
  - `JPG`, `JPEG`, `PNG`, `BMP`, `TIFF`, `WEBP` 형식의 이미지를 열 수 있습니다.
- 4점 선택 기반 투시변환
  - 원본 이미지에서 변환하고 싶은 영역의 꼭짓점 4개를 직접 클릭해 지정합니다.
- 출력 크기 지정
  - 변환 결과의 너비와 높이를 픽셀 단위로 직접 입력할 수 있습니다.
- 이미지 후처리
  - 밝기, 대비, 선명도를 슬라이더로 조정한 뒤 결과에 반영할 수 있습니다.
- 결과 저장
  - 변환 또는 보정이 끝난 이미지를 `PNG`, `JPG`, `BMP` 형식으로 저장할 수 있습니다.
- 상태 안내 및 좌표 확인
  - 상단 상태 메시지와 우측 좌표 표시 영역을 통해 현재 작업 진행 상황을 확인할 수 있습니다.

### 2. 실행 방법

이 프로그램은 Python 기반 GUI 프로그램이며, 실행 전 필요한 라이브러리가 설치되어 있어야 합니다.

#### 실행에 필요한 라이브러리

- `opencv-python`
- `numpy`
- `Pillow`
- `tkinter`

`tkinter`는 Python 설치 환경에 기본 포함되는 경우가 많지만, 환경에 따라 별도 설치가 필요할 수 있습니다.

#### 실행 명령

터미널에서 아래와 같이 실행합니다.

```bash
cd Computer-Vision-Self-Study/1_Perspective-Transform-Image-Processor
python ImageProcessor.py
```

필요 라이브러리가 없다면 아래 예시처럼 설치할 수 있습니다.

```bash
pip install opencv-python numpy Pillow
```

### 3. 프로그램 사용 순서

#### 1) 이미지 열기

- 상단의 `이미지 열기` 버튼을 눌러 작업할 이미지를 선택합니다.
- 이미지가 정상적으로 열리면 원본 이미지가 왼쪽 캔버스에 표시됩니다.

#### 2) 변환할 영역의 4점 선택

- 왼쪽 원본 이미지에서 변환할 영역의 꼭짓점 4개를 순서대로 클릭합니다.
- 클릭 순서는 반드시 아래 순서를 따라야 합니다.

1. 좌상단
2. 우상단
3. 우하단
4. 좌하단

- 점을 잘못 선택했다면 `점 초기화` 버튼을 눌러 다시 지정할 수 있습니다.
- 선택한 좌표는 우측 사이드바의 좌표 표시 영역에서 확인할 수 있습니다.

#### 3) 출력 크기 입력

- 우측 사이드바의 `출력 크기 (픽셀)` 영역에서 결과 이미지의 너비와 높이를 입력합니다.
- 기본값은 너비 `600`, 높이 `400`입니다.
- 원하는 결과 비율에 맞게 숫자를 직접 수정할 수 있습니다.

#### 4) 투시변환 실행

- `변환 실행` 버튼을 누르면 선택한 4개 점을 기준으로 이미지가 직사각형 형태로 보정됩니다.
- 변환 결과는 오른쪽 캔버스에 표시됩니다.

#### 5) 밝기, 대비, 선명도 조정

- 우측 슬라이더를 이용해 결과 이미지를 보정할 수 있습니다.
- 각 슬라이더의 기본값은 `1.0`이며, 의미는 다음과 같습니다.

- 밝기
  - `1.0`은 원본 유지
  - `1.0`보다 크면 더 밝게
  - `1.0`보다 작으면 더 어둡게
- 대비
  - `1.0`은 원본 유지
  - `1.0`보다 크면 대비 강화
  - `1.0`보다 작으면 대비 약화
- 선명도
  - `1.0`은 원본 유지
  - `1.0`보다 크면 더 선명하게
  - `1.0`보다 작으면 더 부드럽게

- 슬라이더 값을 조절한 뒤 `효과 적용` 버튼을 눌러 최종 결과에 반영합니다.
- 처음 값으로 되돌리고 싶다면 `슬라이더 초기화` 버튼을 누르면 됩니다.

#### 6) 결과 저장

- `결과 저장` 버튼을 누르면 현재 결과 이미지를 파일로 저장할 수 있습니다.
- 밝기/대비/선명도 효과까지 적용한 경우에는 최종 보정 결과가 저장됩니다.
- 효과를 적용하지 않았다면 투시변환 결과 이미지가 저장됩니다.

### 4. 사용 시 참고 사항

- 반드시 이미지를 먼저 불러와야 합니다.
- 투시변환을 실행하려면 정확히 4개의 점이 필요합니다.
- 출력 너비와 높이에는 0보다 큰 정수를 입력해야 합니다.
- 이미지 바깥 영역을 클릭하면 점으로 인정되지 않습니다.
- 4개의 점을 모두 선택한 뒤에는 추가 클릭이 제한되며, 다시 선택하려면 `점 초기화`가 필요합니다.
